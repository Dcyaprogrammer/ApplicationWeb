# HCI Process Portfolio

A GitHub Pages website for documenting the iterative design process for the Human-Computer Interaction course project.

---

## 🎨 Website Structure

```
docs/
├── index.html          # Main portfolio page
├── styles.css          # Custom styles
├── script.js           # Interactive scripts
├── assets/             # Images and media
│   └── images/
├── dev.sh              # Development server
└── README.md           # This file
```

## 🚀 Local Development

### Open directly in browser:
```bash
open docs/index.html
# or double-click index.html file
```

### Or use the dev script:
```bash
cd docs
./dev.sh
```

## 📝 Sections

1. **Hero** - Introduction
2. **Brainstorming** - Initial ideas and research
3. **Design Process** - Timeline of design phases
4. **Prototypes** - Iteration showcase (v1.0 → v3.0)
5. **Testing** - User feedback and results
6. **Reflection** - Final thoughts and learnings

## 🎯 Customization

All content areas have placeholder text. Replace with your own:

- Add images to `assets/images/`
- Edit text in `index.html`
- Adjust colors in `styles.css` (`:root` variables)

## 🌐 Deployment

Automatically deployed to GitHub Pages from the `/docs` folder.
